## What is the Peec.ai Looker Studio Connector?

The Peec.ai Looker Studio community connector is a powerful tool that allows you to import your AI search visibility data directly into Google Looker Studio (formerly Google Data Studio). This integration enables you to create custom dashboards, visualizations, and reports using your keyword performance data from AI search engines like ChatGPT, Perplexity, and Gemini.

<Note>
Looker Studio is Google's free data visualization tool that helps you turn your data into informative, easy-to-read, and shareable dashboards and reports.
</Note>

## Key Benefits

- **Real-time Data Access**: Connect directly to your Peec.ai workspace for up-to-date visibility metrics
- **Custom Visualizations**: Create charts, graphs, and tables that match your reporting needs
- **Advanced Filtering**: Filter by prompts, keywords, tags, models, and geographic regions
- **Automated Reporting**: Set up scheduled reports and email delivery
- **Team Collaboration**: Share dashboards with stakeholders and team members

## How to Set Up the Connector

### Prerequisites

Before you begin, ensure you have:

- A Peec.ai account with access to at least one workspace
- A Google account with access to Looker Studio
- Your Peec.ai workspace ID (available in your Peec.ai dashboard)

### Step 1: Access the Connector

1. Open [Looker Studio](https://lookerstudio.google.com)
2. Click "**Create**" â†’ "**Data source**"
3. In the connector gallery, search for "**Peec.ai**"
4. Select the Peec.ai community connector

<Warning>
If you cannot find the Peec.ai connector in the gallery, contact [support@peec.ai](mailto:support@peec.ai) for access.
</Warning>

### Step 2: Authorize the Connection

1. Click "**AUTHORIZE**" to begin the OAuth2 authentication process
2. You'll be redirected to Google's authorization page
3. Sign in with your Google account if prompted
4. Grant the necessary permissions to the connector
5. You'll be redirected back to Looker Studio once authorization is complete

### Step 3: Configure Your Data Source

After authorization, you'll see the connector configuration screen:

#### Required Configuration

**Workspace ID** (Required)
- Enter your Peec.ai workspace ID
- You can find this in your Peec.ai dashboard URL or settings

**Date Range** (Required)
- Looker Studio will automatically provide date range controls
- The connector supports historical data based on your account limits

#### Optional Filters

Use these filters to narrow down your data:

**Prompt IDs**
- Enter comma-separated Prompt IDs to filter by specific prompts
- Example: `prompt123,prompt456,prompt789`
- Leave empty to include all prompts

**Keyword IDs**
- Enter comma-separated Keyword IDs to filter by specific keywords
- Example: `keyword123,keyword456`
- Leave empty to include all keywords

**Tag IDs**
- Enter comma-separated Tag IDs to filter by tags
- Example: `tag123,tag456`
- Useful for organizing keywords by categories

**Model IDs**
- Enter comma-separated Model IDs to filter by AI models
- Example: `gpt-4,claude-3,gemini-pro`
- Leave empty to include all models

**Country Codes**
- Enter comma-separated 2-letter country codes
- Example: `US,GB,DE,FR`
- Leave empty to include all countries

### Step 4: Connect and Validate

1. Click "**CONNECT**" to establish the connection
2. The connector will validate your configuration and fetch sample data
3. You'll see the available fields that can be used in your reports

## Available Data Fields

The Peec.ai connector provides the following data fields:

### Dimensions

| Field Name | Type | Description |
|------------|------|-------------|
| **keyword_id** | Text | Unique identifier for each keyword |
| **keyword_text** | Text | The actual keyword text |
| **date** | Date | Date of the visibility measurement |

### Metrics

| Field Name | Type | Description |
|------------|------|-------------|
| **visibility** | Number | Visibility percentage (0-100) for the keyword on the specific date |

<Info>
The visibility metric represents how prominently your target content appears in AI search results for a given keyword, expressed as a percentage.
</Info>

## Creating Your First Report

### Step 1: Create a New Report

1. After configuring your data source, click "**CREATE REPORT**"
2. Looker Studio will open the report editor with your Peec.ai data source connected

### Step 2: Add Visualizations

**Time Series Chart**
- Drag and drop a "Time series chart" from the toolbar
- Set **Date Dimension**: `date`
- Set **Metric**: `visibility`
- This shows visibility trends over time

**Table View**
- Add a "Table" visualization
- **Dimensions**: `keyword_text`, `date`
- **Metrics**: `visibility`
- Perfect for detailed keyword analysis

**Scorecard**
- Use "Scorecard" for key metrics
- **Metric**: `visibility`
- Apply filters for specific keywords or date ranges

### Step 3: Apply Filters and Controls

Add interactive controls to make your reports dynamic:

**Date Range Control**
- Allows users to change the time period
- Automatically applied to all charts

**Keyword Filter**
- Add a dropdown control for `keyword_text`
- Users can select specific keywords to analyze

**Country Filter** (if using country codes)
- Add dropdown for geographic filtering
- Useful for multi-market analysis

## Troubleshooting

### Common Issues

**"Authorization Failed"**
- Ensure you're using the correct Google account
- Clear browser cache and try again
- Check that your account has access to the workspace

**"No Data Available"**
- Verify your workspace ID is correct
- Check that your date range includes data
- Ensure your filters aren't too restrictive

**"Connection Timeout"**
- Try reducing the date range
- Remove unnecessary filters
- Contact support if the issue persists

**"Invalid Workspace ID"**
- Double-check your workspace ID in the Peec.ai dashboard
- Ensure you have access permissions to the workspace
- Contact your account administrator if needed

### Getting Help

If you encounter issues not covered here:

1. **Contact Support**: Email [support@peec.ai](mailto:support@peec.ai) with:
   - Description of the issue
   - Screenshots if applicable

### Scheduling and Sharing

**Automated Reports**
- Set up scheduled email delivery
- Choose frequency (daily, weekly, monthly)
- Customize recipient lists

**Sharing Options**
- **View Access**: Recipients can interact with the report
- **Edit Access**: Recipients can modify the report
- **Link Sharing**: Generate shareable links with expiration dates
